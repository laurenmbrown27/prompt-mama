# Post Archetypes & Caption Skeletons

Six repeatable shapes. Every caption: hook → tiny story/setup → **the full prompt in
quotes** → payoff → one CTA → 3–5 hashtags. Keyword-rich first line (Instagram SEO).

## 1. Prompt Drop (reel or single image)
**Hook bank:** "This prompt gave me my Sunday back." · "POV: dinner is planned and it's
only 8 AM." · "Steal this — it takes 11 seconds."
**Skeleton:**
> [relatable pain, one line]
> Here's the exact prompt:
> "[full working prompt]"
> Copy it. Paste it. Go sit down.
> Follow @prompt_mama for one prompt a day that gives you your time back. 💗

## 2. Storytime (reel, talking or text overlay)
**Hook bank:** "Weird things I've asked AI as a mom, part __" · "The day AI saved my
[Sunday/road trip/birthday party]" · "I wasn't going to admit this but…"
**Rule:** the mess is the point — never clean it up. End with the prompt that fixed it.
**CTA:** "Drop the weirdest thing you've asked AI 👇"

## 3. The Planner (carousel, 5–7 slides)
**Hook slide:** "5 prompts that plan [X], so you don't have to."
One prompt per slide, big text, brand colors (pink/yellow/blue/navy). Last slide = CTA.
**CTA:** "Save this for [moment]" or comment-keyword for the AI Parent Pack.

## 4. Sunday Reset (reel or photo, Sundays 7 PM)
Soft tone. The week is over, no productivity pressure.
**Skeleton:** one reflective line → one reset prompt or the Reset Journal GPT →
"5 minutes. That's the whole thing. Link in bio." **Hashtag set D.**

## 5. Weekly Recap (reel, alternating Mondays)
"How I used ChatGPT this week:" — 4–5 quick items, at least one funny/weird, at least
one tiny (shows AI is for small stuff too). **CTA:** "Which one do you need? 👇"

## 6. Career Drop (carousel, ≤1/week, Thursdays)
Same Prompt Drop shape, work-flavored. Hashtag set E. Drives to career-ai page.

## CTA rotation (one per post, in order)
follow → save → comment keyword → link in bio → (repeat)

## Comment-keyword automations to set up in Manychat (or reply manually)
- "SUMMER" → AI Parent Pack link
- "RESET" → Reset Journal GPT link
- "CAREER" → career-ai page link
- "GLOW" → Glow-Up quiz link
